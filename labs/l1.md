# Run First Docker(Hello Docker)

## Task

0. Check docker version.
1. Search for image.
2. Pull the docker image.
3. Run the container 
4. Check if it is running or not.
5. Remove all containers

---

## Solution

- Verify the docker version
```bash
docker --version
```

- Search for image.
```bash
docker search akshayithape/hello-docker
```

- Pull the docker image
```bash
docker pull akshayithape/hello-docker
```

- Run the container 
```bash
docker run akshayithape/hello-docker
```

- Check if it is running or not.
```bash
docker ps
docker ps -a
```

- Remove container
```bash
docker rm [container_id]
```

> Note : You can get the container ID from `docker ps` command.
