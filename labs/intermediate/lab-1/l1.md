# Build The Docker Image From Scratch

### Task

1. Write the "Hello-Docker" golang code.(You can use any other language to create binary)
2. Build the code and Run to verify.
3. Write the dockerfile to create image.
4. Build the dockerfile.
5. Login to Docker Hub.
6. Push the docker image to Docker Hub.

### Solution

- Write the "Hello-Docker" golang code.(You can use any other language to create binary)

Ref : hello-docker.go

```go
package main

import "fmt"

func main(){
	fmt.Println("Hello Docker")
}
```

- Build the code and Run to verify

```bash
go build hello-docker.go
./hello-docker
```

- Write the dockerfile to create image

Ref : dockerfile

```dockerfile
FROM scratch
Add hello-docker /
CMD ["/hello-docker"]
```

- Build the dockerfile

```bash
docker build -f dockerfile -t akshayithape/hello-docker:v1 .
```

> Note : \
  After `-f` give the path docker file. \
  For tag `-t` give your dockerhub username. \
  After `/` give your docker image name. \
  After `:` you can give version number(Optional).

- Login to Docker Hub

```bash
docker login
```

> Note : Enter your Docker Hub Username and Password.

- Push the docker image to Docker Hub

```bash
docker push akshayithape/hello-docker:v1
```
